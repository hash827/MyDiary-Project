# MyDiary-Project

1. Create
 - 다이어리의 날짜, 날씨, 장소, 제목, 이미지, 내용, 카테고리, 기분, 시간관리를 입력하여 다이어리를 생성


2. Search
 - 다이어리의 제목이나 날짜를 이용해 검색 가능


3. Delete
 - 리스트에서 해당 다이어리를 길게 클릭하여 다이어리 삭제 가능
 
 
4. Update
 - 리스트에서 해당 다이어리를 클릭해서 다이어리 수정 가능
 - 이미지를 클릭하면 이미지 수정 가능
